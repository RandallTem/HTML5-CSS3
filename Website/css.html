3. Каскадные таблицы стилей CSS3

CSS (Cascading Style Sheets) - каскадные таблицы стилей, используются для описания внешнего вида документа, написанного языком разметки.

Каскадные таблицы стилей описывают правила форматирования элементов с помощью свойств и допустимых для них значений. Для каждого элемента можно использовать ограниченный набор свойств, остальные не будут оказывать на него никакого влияния.

Преимущества примения стилей:

1. Разграничение кода и оформления

2. Разное оформление для разных устройств

3. Расширенные способы форматирования

4. Ускорение загрузки сайта

5. Единое стилевое оформление множества документов

6. Централизованное хранение

Типы стилей

Стили представляют набор параметров, управляющих видом и положением элементов веб-страницы.

Существует несколько типов стилей:

- Стиль браузера - оформление, которое по умолчанию применяется к элементам веб-страницы браузером.

- Стиль автора - стиль, который добавляет к документу его разработчик.

- Стиль пользователя - стиль, который может включить пользователь сайта через настройки браузера. Такой стиль имеет более высокий приоритет и переопределяет исходное оформление документа.

Способы добавления стилей на страницу

1. Inline или внутренние стили подключаются с использованием атрибута style.

2. Глобальные стили устанавливаются тегом <style>, который должен быть размещен в теге <head>.

3. Тегом <link> в атрибуте href задается путь к файлу со стилями. Файл должен иметь расширение .css. Тег <link> должен быть размещен в теге <head>.

Селекторы

Селектор элемента

Если в качестве селектора выступает имя элемента (тега), то в этом случае все элементы данного тега будут соответствовать заданным стилям. Если внутри этих тегов будут находиться текстовые элементы, они будут наследовать определенные правила.

Селектор класса

Если необходимо задать стили для индивидуального элемента или задать разные стили для одного тега, следует использовать селектор класса. Имена классов должны начинаться с латинского символа и могут содержать в себе символ дефиса и подчеркивания.

Чтобы привязать к элементу класс, необходимо в открывающем теге задать атрибут class и присвоить ему имя класса, описанного в стилях.

Селектор идентификатора

Идентификатор - особый вид селектора, которым помечается один элемент на странице. Чтобы указать селектор идентификатора, в CSS необходимо использовать конструкцию #имя_идентификатора.

Чтобы привязать к элементу идентификатор, необходимо в открывающем теге задать атрибут id и присвоить ему имя идентификатора, описанного в стилях.

Работа со шрифтами

Font - универсальное свойства, которое позволяет устанавливать одновременно несколько характеристик шрифта.

Font-Family - устанавливает семейства шрифта для оформления текста содержимого.

Font-size - определяет размер шрифта элемента.

Font-stretch - устанавливает узкое, нормальное или широкое начертание шрифта.

Font-style - задает начертание шрифта.

Font-variant - определяет как представлять строчные буквы.

Font-weight - устанавливает насыщенность шрифта.

Работа с цветом

Background-color - позволяет задать элементу фоновый цвет. Цвет можно задавать несколькими способами:
- шестнадцатеричным значением
- функцией RGB(red, green, blue)
- функцией hsla(hue, saturation, lightness, opacity).

Background-image позволяет задать фоновое изображение для элемента.

Border позволяет установить границу вокруг элемента. Значение следует задавать в следующем порядке:
- border-width - толщина границы.
- border-style - стиль линии.
   -  dotted - точки
   - dashed - пунктир
   - solid - сплошная
   - double - двойная
   - groove - втиснутая
   - ridge - выпуклая
   - inset - внутрь
   - outset - наружу
- border-color - цвет линии.

Outline - позволяет установить внешнюю границу элемента на всех четырех сторонах одновременно. В отличие от рамки, контур размещен поверх блока элемента и не влияет на размер, но может перекрывать внешний отступ.

Псевдоклассы и псевдоэлементы

Псевдоэлементы - элементы CSS стилей, который позволяют задать стиль элементов, не определенных в дереве элементов документа, а также генерировать содержимое, которого нет в исходном коде текста.

Псевдоклассы - элементы CSS стилей, которые определяют динамическое состояние элементов, которое изменяется при взаимодействии пользователя со страницей.

Псевдоклассы

:link, :visited, :hover, :active

:link - задает стили для не посещенной ссылки.

:visited - задает стили для ссылки, по которой уже выполнялся переход.

:hover - задает стили для элемента, когда на него наводят курсор. Действует не только для ссылок.

:active - задает стили для ссылки на которой удерживается нажатый курсор.

Работа с текстом

Выравнивание текста

Свойство text-align определяет горизонтальное выравнивание текста внутри элемента.
Имеет следующие значения:
- center - вырвнивание текста по центру.
- justify - выравнивание по ширине.
- left - по левому краю.
- right - по правому краю.
- start - аналогично left, если текст идет слева на право, или right, когда тект идет справа на лево. CSS3.
- end - такое же как start, только наоборот. CSS3.

Текст в параграфе

Оформления текста

Свойство text-decoration позволяет задать оформление тексту. Имеет следующие значения:
- line-through - перечеркнутый текст.
- overline - линия над текстом.
- underline - подчеркивание.
- none - отменяет все эфекты, в том числе подчеркивание у ссылок.

Текст в параграфе

Свойства text-overflow, overflow и white-space

Свойство text-overflow определяет видимость текста в блоке, если текст целиком не помещается в заданную область. CSS3. Работает только в том случае, если задано свойство overflow. Имеет следующие значения:
- clip - текст обрезается по размеру области.
- ellipsis - текст обрезается и к концу строки добавляется многоточие.

Свойство overflow задает отображение содержимому блочного элемента, если оно целиком не помещается и выходит за область заданных размеров. Имеет следующие значения:
- visible - отображает все содержимое, даже за пределами области.
- hidden - отображается только то содержимое, которое не выходит за пределы области.
- scroll - добавляются полосы прокрутки.
- auto - полосы прокрутки добавляются только при необходимости.

Свойство white-space устанавливает отображение пробелов между словами. Имеет следующие значения:
- normal - тест выводится обычным образом, переносы строк устанавливаются автоматически.
- nowrap - пробелы не учитываются, переносы строк игнорируются, и весь текст отображается в одной строке.
- pre - текст отображется с учетом всех пробелов и переносов.
- pre-line - пробелы не учитываются, текст автоматически переносится на новую строку, если не помещается в заданную область.
- pre-wrap - пробелы и переносы в тексте сохранются, но если строка не помещается в заданную область, то текст автоматически переносится на следующую строку.

Работа с таблицами

Свойства border, bolder-collapse, border-spacing, empty-cells

Свойством border можно задать границы таблице.

Свойство border-collapse пощволяет задать отображение границы вокруг ячеек таблицы. Имеет несколько значений:
- collapse - рамки ячеек сливаются в одну, а промежутки между рамками убираются.
- separate - рамки ячеек располагаются отдельно.

Свойство border-spacing позволяет задать отступы между ячейками и рамкой таблицы. Если задано одно значение - отступы со всех сторон одинаковые, если два значения, то первое задает расстояние по горизонтали, второе - по вертикали.

Свойство empty-cells позволяет задать параметры отображения пустых ячеек. Имеет два значения:
- show - рамка и фон отображаются.
- hide - скрывает ячейку, задавая ей свойство display: none.
